# 基本语法

## 注释

被JavaScript引擎忽略的部分叫做注释，它的作用是对代码进行结束。JavaScript提供的注释的写法如下：

```javascript
// 单行注释
/*
 多行
 注释
*/
```

## 变量

JavaScript声明变量需使用关键字 `var` 来声明：

```javascript
var i;
var sum;
// 或者
var i, sum;
// 声明变量时赋值
var message = "Hello";
var i = 0, j = 0, k = 0;
// 变量赋值不写 var 也是可以的
a = 1;	// 等同于 var a = 1; 但这样写法不利于表达，且容易不知不觉地创建全局变量
```

如果未在 `var` 声明语句时指定初始值，初始值会默认为 `undefined`。
